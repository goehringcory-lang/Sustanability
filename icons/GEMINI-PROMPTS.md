# Gemini Image Prompts — Sustain Icon Pack

Use these prompts in Gemini (or Midjourney, DALL-E, etc.) to generate richer versions of the 5 category icons. Each prompt maintains the same visual language so the set feels cohesive.

---

## Master Style Directive

Paste this before each individual prompt to keep the style consistent:

> **Style:** Minimalist line icon on a pale granite-textured circular background. Thin, precise strokes in near-black (#0F0F0F). One accent detail in warm Yosemite rust (#A85121). The background circle should have a subtle cool-stone texture like polished El Capitan granite — pale gray-white with faint mineral grain. No gradients on the icon itself. No drop shadows. No fills except the accent color. The line weight should feel like an architect's pen — confident, clean, geometric. Square 1:1 format, 1024x1024px, transparent or white outer background. The icon should feel like it belongs on an Apple product page or an Aesop store window.

## Tips for Best Results

- Generate at 1024x1024 and then scale down
- If the output is too detailed, add "even more minimal, fewer strokes, less detail" to the prompt
- If the granite texture is too heavy, say "barely visible stone grain, almost smooth"
- Request PNG with transparent background if your tool supports it
- The rust accent color is #A85121 — if the AI picks a different warm tone, regenerate or color-correct in any image editor
